How it works

We use Cordova to build installable apps from HTML files. All we need you to do, is give us a zip file containing an index.html file (along with your other assets and HTML). Once you have uploaded the zip, we create a temporary Cordova skeleton app for your content.

What's else can I do?

There are various ways you can customise your apps:

  • Put an icon.png file in the root
  • Add config.json
  • More docs on config.json coming soon - but it can do everything possible
